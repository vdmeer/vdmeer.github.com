Development and Open Source


Github

Here are a few repositories for my open source projects. All of them are hosted now on GitHub (as is this single page). The projects are:

  • skb-core - SKB core/dist databases, documentation and Wiki
  • skb-locale - Gnu Gettext pot and po files for SKB localisation
  • skb-php - The PHP port of the SKB
  • skb-java - SKB Java Port
  • skb-latex - LATEX class for organising documents, can be integrated into the SKB
  • a3ds-java - Java implementation of A3DS, including compilers for contract and policy, utils, types and tribe



RINA/TINOS

I did not directly work on the implementation, but participated in some work for the TINOS platform, a virtualised networking platform based on Eclipse Virgo, originated in the FP7 project 4WARD and later adopted by the Pouzin Society to protype RINA. TINOS is hostedon GitHub.
