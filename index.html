Development and Open Source


Github

Here are a few repositories for my open source projects. All of them are hosted now on GitHub (as is this single page). The projects are:

  • skb - Java and PHP implementation including a type library (Java), a compiler framwork for ANTLR (Java), some languages (Java), the Knowledge Base (PHP, Java) and some other projects
  • skb-latex - LATEX class for organising documents, can be integrated into the SKB
  • skb-dependencies - some dependencies for the SKB (jar files that are not found on common repositories)



RINA/TINOS

I did not directly work on the implementation, but participated in some work for the TINOS platform, a virtualised networking platform based on Eclipse Virgo, originated in the FP7 project 4WARD and later adopted by the Pouzin Society to protype RINA. TINOS is hostedon GitHub.

If you are looking for code analysis software, the "Structure Guys" at Headway Software have a product family called Structure 101. I am using their Java version right now and experimenting with the PHP variant. Structure 101 is great for defining an architecture and structuring all code accordingly. Follow the link with the image for details. Note: licenses for open source projects are for free (that's how I got mine)!
